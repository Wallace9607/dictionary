# My Vocabulary in English

Aplicação web para estudo de vocabulário em inglês.

## Funcionalidades

- Adicionar frases em inglês
- Registrar pronúncia aproximada
- Adicionar exemplos de palavras
- Ouvir pronúncia das frases
- Testar pronúncia usando reconhecimento de voz
- Armazenamento local com LocalStorage

## Tecnologias

- HTML5
- CSS3
- JavaScript
- Web Speech API
